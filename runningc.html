Running C Programs.

Running C Programs in Windows 10

  • Step 1: Download " DOSBox " from Here.
    Download "Turbo" Folder from https://goo.gl/3m1JBr
  • Step 2: Install DOSBox.
  • Step 3: Extract "Turbo" folder into C:\ ( eg: c:\Turbo )
  • Step 4: After that click on the DOSBox shortcut that is created at the desktop.
  • Step 5: Write these command:

    mount c c:\Turbo (After this you will see a message that Drive C is mounted as a local directory C:\Turbo)
    c: (Switch to C directory)
    cd tc (Go to TC folder)
    cd bin (Go to Bin folder)
    tc (To execute Turbo C or Turbo C++)

  • ***** Now Turbo C or C++ is running and you can write your programs. *****


Possible Issues

(1) Error : When compiling program.You get the error "Unable to open stdio.h in Turbo C !"

In Turbo C, goto OPTIONS > DIRECTORIES, try changing c:\tc\include to c:\include or change it according to the drive letter you mounted on the DOSBOX.


(2) Shortcut issue

Ctrl+F9 is a shortcut for DOSBox to exit. It is also the shortcut to compile C Program. So if you use shortcuts then there would be an issue. If you face such problems, follow these steps:


(i). When you are running Turbo C++ press Ctrl+F1.

(ii). Click on Shut Down.

(iii). Click on Del.

(iv). Click on Add.

(v). Press q and then click on mod1 shown at left bottom corner (It sets the Ctrl+q as a shortcut to exit from DOSBox).

(vi). Click Save and then Exit.

Now you can compile your programs using Ctrl+F9 and use Ctrl+q to exit from DOSBox.
